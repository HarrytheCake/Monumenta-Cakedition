&bAurora Rage+ Mode Changes:&r
Based on player feedback for Aurora's Rage+ Mode, the following
changes have been made:
• Rage+ mode is now a single-life experience. If your team wins
the fight, all players (dead or not) will be restored to life as
normal, but if a team fully wipes, the instance is ended and
Aurora can no longer be challenged.
• In the case of fullwipes, your team's reward payout is lowered
to be equivalent to that of a 100% rage win • so think carefully
about if you can take on that next rage challenge!
• The reward payout for Rage+ mode has been adjusted to
compensate for the increase in risk that now comes with the mode
• The Highest Rage Achieved leaderboard has been reset following
these changes, with any player who previously had >100 Rage now
being set to exactly 100.

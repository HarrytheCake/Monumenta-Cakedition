### Happy Depths-versary!
Hard to believe it, but it's Darkest Depths 5th birthday! (July
17, 2021) And with Monumenta celebrating it's 10th anniversary
this year, that means it's almost the midpoint in our history..
in any case, 6-player mode will be making another return! Start
a run with up to five other friends and compete for the special
leaderboard spots for the next two weeks. The event will run
until the weekly update of July 29th. Good luck and have fun!
### Skill Descriptions:
• Skill descriptions have received a backend refactor! This
should not affect any base functionality, but report any issues
if you see them! There are also some new changes as well:
  • Added a system to display extra info for certain abilities
in the class GUI! They can be viewed by pressing F (swap hands)
over the ability icon, and contain auxiliary info such as:
region scaling, applying aspects, having extra customizable
triggers, and explaining keywords or other complex mechanics
    • For resource pack users, there is also a new feature that
will highlight the abilities with extra info when in viewing
mode
  • Region-scaled values are now colored light yellow, light
blue, or light pink depending on which region's value is
currently being displayed
  • Class/spec blurbs in their descriptions now are center-
aligned
  • Center-aligned text is now always perfectly aligned instead
of being slightly off
  • Changed the default appearance of descriptions to have a new
style of dashed line and slightly more padding
  • Changed list items to use a bullet icon instead of a dash
  • A plentitude of skills have had their descriptions adjusted,
fixing typos, poor line breaks, incorrect level behavior, or
flipped charm coloring### Minor New Features:
• Added a new epic cosmetic skill set for Scout, which can be
obtained from a certain NPS after breaking into the Cradle's
weapons factory
• The Light Blue lobby has been fully rebuilt, thanks to Sw0ut!
